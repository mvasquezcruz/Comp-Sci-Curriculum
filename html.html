HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating webpages. HTML consists of a series of elements which tell the browser how to display the webpage content.

HTML Element

An HTML element is defined by a start tag, content in between and an end tag. The element is everything from the start to end tag. For example:

< h1 > Header < /h1 >

The start tag will always be the defined element in <> and the ending tag the same but with a / , < / >.

This is a list of HTML elements you will use frequently when writing up an HTML page:

HTML Element What It Means
< !DOCTYPE html > Defines that the page is an HTML5 document
< html > Element is the root element of an HTML page
< head > Element which contains meta (data about data) information (title, CSS, scripts, styles, ect.) about the HTML page
< title > Element specifies the title for the HTML page, which will be name for the HTML shown in the browser bar or in the page's tab. Placed insie head element
< body > Element defined the pages body and it contains all visible content of the page such as heading, paragraphs, images, links, tables, lists, etc.
< h(1-6) > Element defines headings. h1 being the largest heading and h6 being the smallest
< p > Element defines a paragraph
< br > Element defines a break or space in the HTML
< a > Element defines a hyperlink
< img > Element embeds an image (does not require an end tag)
< div > Element defines a division or section in the HTML

HTML Attributes

HTML atrributes ptovide additional information about the element. Attribute are specifed in the start tag and usually come in name/value pairs ( < start tag name = "value"> < /ending tag >)

href Attribute

As you know, the < a > starting tag defines a hyperlink, the href attribute specifies the URL of the page the link goes to. The text between the start and ending tag will be the hyperlink the viewer can see. For example:

< a href = "URL"> Link < / a >

src, align, height, & width Attribute

The < img > element is used to embed an image to the HTML, the src attribute specifies where the image displayed is from. When displaying images from the internet, you can use the URL (Image Adress) of the image. For example:

< img src = "URL" >

The image tag can also contain the align, height, and width attributes. Align specifies where in the HTML will the image be dispalyed on the screen (left, center, right) and height and width specify the height and width of the image. For example.

< img src = "URL" width= "#" height= "#" align= "left/center/right">

style Attribute

The style attribute adds styles to an element, such as color, font, size, background color, etc. For example:

< p style = "CSS property: CSS value" > < /p >

CSS, or Cascading Style Sheets, is the language used to style HTML. When using the style attribute the CSS property is what you'd like to style and the value is the specific change. For example if you'd like to change the background-color (CSS property) to blue (CSS value).

Below is a table of CSS properties:

CSS Property What It Means
Background-color Defines the background color of an element
Color Defines the color of the text in the element
Font-family Defines the font to be used in the element
Font-size Defines the text size in an element
text-align Defines where in the HTML will the text be displayed on the screen.

You can use mutliple CSS properties to style an element just separate properties using a semicolon (;). For example:

< h3 style = "background-color: #87CEEB ; color: #FFFF00; font-family: Arial Rounded MT Bold; text-align: center " > < /h3 >

HEX Colors

In HTML, a color can be specificed using a hexadecimal (6) value. HEX colors are written in the form:

#RRGGBB

RR stands for red, GG stands for green, and BB for blue. These are hexademical values between 00 and FF. FF is the highest value and 00 being the lowest value.

For example, #FFO000 would be displayed as red, red being set to the highest value and green and blue being set to 00. You can use different values of red, blue, and green to make the color of preference.

Below is a list of colors and their HEX values:

Color HEX Value

Pink #FFC0CB
Orange #FFA500
Yellow #FFFF00
Green #00FF00
Blue #0000FF
Indigo #4B0082
Violet #8F00FF
Black #000000
White #FFFFFF

HTML Tables

One way to organize data on an HTML page can be with a table. HTML tables consists of table cells inside rows and columns. Making tables on HTML is from top to bottom. The data you insert in each table row working from left to right.

Making a table you will begin with the < table > tag and inside of that tag you will begin making table rows < tr >. In the first < tr > you'd begin with the headers of the columns < th >. Inside the other table rows tags do you write your data for each column. < td >. For example:

< table >

< tr >

< th > Land Animal < /th >

< th > Sea Animal < /th >

< /tr >

< tr >

< td > Lion < /td >

< td > Whale < /td >

< /tr >

< tr >

< td > Horse < /td >

< td > Dolphin < /td >

< /tr >

< tr >

< td > Dog < /td >

< td > Fish < /td >

< /tr >

< /table >

This table would look something like this:

Land Animal Sea Animal
Lion Whale
Horse Dolphin
Dog Fish

HTML Lists

Another way to organize data on HTML is with lists. There are ordered/numbered lists < ol > and underored/bulleted lists < ul >.

To make either type of list, start with the tag and insert the list item with the < li > tag. For example:

Ordered List

< ol >

< li > Coffee < /li >

< li > Tea < /li >

< li > Milk < /li >

< /ol >


Unordered List

< ul >

< li > Coffee < /li >

< li > Tea < /li >

< li > Milk < /li >

< /ul >

Those lists would look like this:

  1. Coffee
  2. Tea
  3. Milk
  • Coffee
  • Tea
  • Milk

HTML Formatting

All HTML pages must start with the < !DOCTYPE html > declaration, followed by the < html > tag. Inside the html tag you can begin with a and then a < body >. This should be the layout you start ALL your HTML pages with:

< !DOCTYPE html >

< html >


< head >

(here you can add links, styles, title, meta)

< /head >


< body >

(here you can add paragraphs, headers, images, links, tables, lists, etc.)

< /body >


< /html >
