Bash

Bash is a command language. It is commonly used as the default login shell language for most Linux servers. You can use Bash to efficiently manipulate, access or move files and directories.

To navigate our class server, you will need to know these common bash commands:

Command What It Means
w Prints a list of who is currently logged in the server
cd Changes working directory to home directory
cd folder Changes to the specified directory ('folder')
pwd Prints the working directory (the folder you are currently in)
ls Prints a list of files in the current directory
ls -a Prints a list of files in the current directory, including hidden files
ls -l Prints a list of files in the current directory, including permissions and other information
mkdir folder Create a new directory called "folder"
rm folder Removes a file ('folder')
cp Copy a file
mv Move a file
cat Prints the contents of file
head Prints the first ten lines of file
tail Prints the last ten lines of file
passwd Change your password
date Print the current date
cal This month's calendar
history Print a history of the commands that have been run
whoami Tells you what user you are logged in as
