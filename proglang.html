Programming Languages

At the beginning of quarter 3, as a class you will choose one of three popular programming languages: Python, Javascript or C++.

Each programming language of high demand and used for different purposes. Below on this table you can see the pros and cons of each programming language. There are also links which can give you a deeper look into what coding with that language might look like.

Python Javascript C++
Common Uses Artificial intelligence,financial services, data sciences, applications, etc. Business applications and android mobile applications. Computer games and mathematical simulations.
Pros It is a widely regarded easy to learn language. It has simple synthax, large library, and many toolkits. Used by large businesses around the world and commonly associated with client-server applications. Works well for programming the systems that run applications, as opposed to the applications themselves. Also works well for multi-device and multi-platform systems.
Cons Not suitable for mobile applications. Javascript is not ideal for applications that run on the cloud as opposed to servers. C++ has a complex synthax which can make it difficult for new programmers.
