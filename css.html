Cascading Style Sheets (CSS)

As you learned through the HTML style attribute, CSS is used to format and style the layout of page.

There are 3 ways to add CSS to an HTML document:

  • Inline
  • Internal
  • External

Inline CSS

An inline CSS uses the style attribute of an HTML element to style the single element. For example:

< h3 style = "color: yellow (#FFFF00)"> A Yellow Heading < /h3 >

A Yellow Heading

Internal CSS

An internal CSS uses the style element, defined in the < head > section of the HTML page This is used to define the style for the single HTML page. It would look something like this:

< !DOCTYPE html >

< html >


< head >

< style >

body {background-color: green (#00FF00);}

h1 {color: red (#FF0000);}

p {color: blue (#0000FF);}

< /style >

< /head >


< body (green background)>

< h1 > This header has red text < /h1 >

< p > This paragraph has blue text < /p >

< /body >


< /html >

External CSS

An external CSS is used to define the style of many HTML pages. External CSS is a link placed in the < head > section of each HTML page. The external CSS would be only CSS properties and values saved with a .css extension.

For example:

HTML

< !DOCTYPE html >

< html >


< head >

< link rel = "stylesheet" href = "styles.css" >

< /head >


< body >

< h1 > This is a header < /h1 >

< p > This a paragraph < /p >

< /body >


< /html >

"styles.css"

body {

background-color: green (#00FF00);

}

h1 {

color: red (#FF0000);

}

p {

color: blue (#0000FF);

}
